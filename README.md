碧之轨迹自动钓鱼

基本原理是不停的截屏看固定区域内有没有感叹号颜色的像素出现，有就发一个2键。

首先需要在config.ini里设置好游戏路径(我懒的写一个选择路径的流程了)。

目前的程序是按照960*544游戏分辨率来做的，如果分辨率不一样，可以在place.ini里进行修改，现有的place.ini只有租船小屋和间道浅滩尽头两个地点，可以自行增加，程序会自动加载。

place.ini的写法是
<钓鱼点名称>=<矩形左上角X坐标>,<矩形左上角Y坐标>,<矩形长度>,<矩形宽度>

注意这个X坐标和Y坐标是针对游戏窗口左上角的，程序会自动把游戏窗口的位置计算进去(注意开始钓鱼之后就不会重新算了，这个时候不要挪窗口)。

打开程序选好钓鱼点和次数，在游戏中开始钓鱼，选好钓竿，把选择框放到需要的鱼饵上，点"开始"进行钓鱼。
